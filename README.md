# Legged Robot for Locomotion in Rough Terrain with Perception-Based Posture Adaptation

This is the repository for the capstone project of Computer Engineering undergraduate Hania Kashif and Computer Science undergraduates Raahim Hashmi and Syed Zuhair Abbas Rizvi from Habib University. The project is co-supervised by Dr. Basit Memon and Tabsheer Ali Askari.

## Project Statement 

Our goal is to make the T-Hex legged robot kit capable of traversing rough terrain and adapt its posture to move through confined spaces and avoid obstacles

## Repository Structure

- `Code/`: Contains all our code files related to hardware, simulation, and analysis.
- `Documents/`: Contains all technical documents and diagrams related to the project Our Capstone submission documents can be found in our shared [Overleaf Project](https://www.overleaf.com/1921852215dqddskmyvwcp#9cba15).
- `Resources/`: Constains resources, for example datasheets, CAD files, simulation models, etc.
- `Archive/`: Contains old versions or backups of code files, documents, models, etc.
- `Demo/`: Contains videos showcasing our project at various stages.
